# lesson22.05
Подключение пакета Microsoft.Data.Sqlite
В Visual Studio:
Меню Project → Manage NuGet Packages…
Вкладка Browse, в поиске набери: Microsoft.Data.Sqlite
Выбери пакет Microsoft.Data.Sqlite (от Microsoft) и нажми Install.
Либо через Package Manager Console:
Install-Package Microsoft.Data.Sqlite
